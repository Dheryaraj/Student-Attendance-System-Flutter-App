
Project Overview
This project is developed using Flutter as part of academic coursework. It implements a simple Student Attendance System that allows users to add student names and mark their attendance as Present or Absent.
The application demonstrates the use of Flutter widgets, state management, and dynamic data display using ListView.

Objectives
The main objectives of this project are:
To implement user input handling in Flutter
To manage state using StatefulWidget
To dynamically update UI components
To display data using ListView

Technologies Used
Flutter
Dart
Features
Add student name
Mark attendance as Present or Absent
Real-time UI update
Dynamic ListView display
